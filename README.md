## Structura

```
alessiajuglan-nailtech/
├── index.html           # Acasă
├── despre.html          # Despre Alessia
├── servicii.html        # Toate serviciile + FAQ + regulament
├── galerie.html         # Galerie cu filtre
├── recenzii.html        # Recenzii clienți
├── programare.html      # Booking flow 5 pași
├── contact.html         # Contact + formular + program + hartă
├── styles.css           # Stilizare completă (premium/luxury)

├── services-data.js     # Date servicii

```

## Booking online

Flow integral în 5 pași:

1. **Serviciu** — alege din 11 servicii
2. **Data** — calendar interactiv (L–V, fără zile trecute)
3. **Ora** — intervale generate dinamic pe baza duratei și programului
4. **Datele tale** — nume, telefon, email, mențiuni
5. **Confirmare** — rezumat complet
