# Alessia Juglan — Nail Artist

Website premium pentru Alessia Juglan Nail Tech — manichiură, construcție gel,
design 3D și French Glass în Iași.

## Structura

- `index.html` — pagina principală
- `styles.css` — stilizare (luxury / premium)
- `script.js` — interactivitate (meniu, filtre servicii, status program)
- `assets/` — iconițe și logo

## Rulare locală

Deschide `index.html` direct în browser, sau:

```bash
python3 -m http.server 8080
```

și accesează `http://localhost:8080`.
